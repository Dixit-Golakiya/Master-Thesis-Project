# Firmware — STM32G474RE

C firmware written in **STM32CubeIDE** using the HAL library, targeting the
**NUCLEO-G474RE** board. Implements a six-state finite state machine (FSM)
for galvanostatic discharge measurement and real-time ESR/capacitance
extraction (thesis Section 3.3.8–3.4).

## Contents (place your CubeIDE project export here)

```
stm32/
├── project-name.ioc            # CubeMX config — regenerate base code from this
├── Core/
│   ├── Inc/                    # headers, incl. capacitance_analysis.h
│   └── Src/                    # main.c, capacitance_analysis.c, etc.
├── Drivers/                    # HAL/CMSIS drivers (as generated by CubeMX)

```

> Keep the `.ioc` file — anyone can regenerate the peripheral init code from
> it via STM32CubeMX. Do **not** gitignore `Core/Inc` and `Core/Src` — your
> actual application logic lives there.

## Firmware architecture (FSM)

```
Initialisation → Idle → Pre-trigger Acquisition → Discharge
              → Parameter Extraction → Result Transmission → (back to Idle)
```

State transitions are driven by TLV3541 comparator hardware interrupts and
ADC DMA callbacks. See thesis Fig. 3.8 / Table 3.7 for full detail.

## Peripheral configuration

- **ADC1/ADC2/ADC5** — 12-bit, 32× oversampling (5-bit right shift), 14
  channels total, TIM3-triggered at 1 kHz, DMA circular double-buffered.
- **TIM3** — 1 kHz synchronised trigger for all 3 ADCs.
- **TIM2** — free-running 64-bit timestamp counter.
- **TIM4** — PWM control of the discharge pulse envelope.
- **DAC1 Ch2** — sets the current sink reference (0 at startup).
- **UART2** — 100000 baud, 8N1, result/CSV reporting.

## Core algorithms

- **ESR** — `ESR = |V(t0⁻) − V(t0⁺)| / Idischarge`, where `V(t0⁻)` is a
  10-sample (10 ms) pre-trigger mean and `V(t0⁺)` is the first stable
  post-commutation sample.
- **Capacitance** — least-squares linear regression over a sliding window
  (500 ms to 0.9×tend, 50 ms steps) of the discharge slope, scored by a
  4-tier R²/RMSE quality criterion (see thesis Table 3.9). Per-cell values
  recovered via the series-difference method.
- **Safety** — per-sample cell voltage check
  `(Vk − Vk+1) < Vthreshold` for all 8 nodes; immediate DAC=0, PWM stop, and
  fault LED on trip. Independent hardware emergency stop via button S3.

## Build / Flash

1. Open **STM32CubeIDE**.
2. `File → Open Projects from File System...` → select this `stm32/` folder.
3. If regenerating peripheral init: open the `.ioc` file and let CubeMX
   regenerate — application code in `Core/Src` (state machine, analysis
   engine) is preserved separately from generated init code.
4. Build (`Ctrl+B`), then flash to a connected **NUCLEO-G474RE** via the
   onboard ST-LINK (`Run → Debug` or `Run → Run`).
5. Connect to UART2 at **100000 baud, 8N1** to receive per-cycle results
   (per-cell capacitance, ESR from all sensors, R², RMSE).
